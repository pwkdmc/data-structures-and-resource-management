# Структуры данных и управление ресурсами в C++
## Описание проекта
Этот проект представляет собой сборник из пяти консольных приложений на C++, реализующих различные функциональности для работы с данными, шифрованием, управлением библиотекой и игровыми механиками.
## Список приложений
### 1. Умный анализатор успеваемости (SmartPerformanceAnalyzer)
- Ввод данных о студентах и их оценках по предметам
- Расчет среднего балла по каждому предмету
- Рейтинг студентов по успеваемости
- Выявление проблемных предметов
- Табличный вывод результатов
### 2. Текст-шифровальщик (Ciphertext)
- Шифрование текста шифром Цезаря (поддерживает русские и английские буквы)
- Статистика символов, слов, предложений и цифр в тексте
- Поиск палиндромов
- Анализ частоты встречаемости элементов
### 3. База данных студентов (StudentDatabase)
- Добавление, просмотр и поиск студентов
- Хранение информации: имя, возраст, предметы и оценки
- Экспорт данных в файл
- Статистика по возрастам и средним баллам
### 4. Система управления библиотекой (LibraryManagementSystem)
- Учет книг с полями: автор, название, год, жанр, доступность
- Поиск книг по названию, автору и жанру
- Статистика распределения по жанрам, годам и доступности
- Добавление новых книг
### 5. Игра на память (MemoryGame)
- Игра "Найди пару" на поле 4×6
- Динамическое перемешивание карт
- Подсчет очков
- Графическое отображение игрового поля в консоли
## Особенности реализации
- Поддержка русского языка в консоли
- Обработка некорректного ввода
- Использование стандартных контейнеров STL (vector, map, set)
- Модульная структура с разделением на заголовочные и исходные файлы
- Интерактивное меню с возможностью повторного выполнения задач
## Проект демонстрирует различные аспекты программирования на C++:
- Работу со структурами данных
- Обработку строк и текста
- Файловый ввод-вывод
- Интерактивные консольные интерфейсы
- Игровую логику
